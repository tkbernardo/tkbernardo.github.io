Methodology

As a Human-Computer Interaction (HCI) practitioner I rely upon a system of methods for investigating new problem spaces, validating design decisions, and testing for usability issues. Here is a collection of methods I've used on my projects:

Wireframing

Useful in ideating new features or systems. Used to communicate early, low-fi design of layout and structure.

Prototyping

Paper prototypes are a low cost way to test for usability issues, while hi-fi interactive prototypes are useful testing as well as communicating with developers.

User Testing

Crucial element of UX Design for uncovering usability issues, as well as a means to collect qualitative and quantitative data.

Interviews

Helpful for finding out how users goals, needs, and pain-points, as well as how they interact with a system.

Contextual Inquiry

Valuable tool for learning about users in their natural settings. Can gain completely new insights which were not previously thought about.

Observations

Useful for gaining insights into a new domain. Can collect a large volume of data, unbiased by the Hawthorne effect.

Heuristic Evaluation

Guidelines, popularized by Jakob Nielsen among others, which are useful in finding usability and accessibility issues.

Competitive Analysis

Process of analyzing competitors in a particular domain, to learn how they handle common design problems.

Storyboarding

Visual tool for conceptualizing a design solution through storytelling, using quick sketches.

Card Sorting

Technique, which can be conducted on-line or in person, for creating understandable groupings of information. Useful in designing an information architecture, and menu structures.

Journey Mapping

Used to plot out an ideal user's experience using a service or a system. Good for visualizing pain-points and opportunities for improvement of the user experience.

Affinity Diagramming

Process of grouping insights from observations into insight clusters, to gain a understanding of a particular domain. Useful for generating User Stories.

Sketching

Quick and easy way to generate initial design ideas. Very low cost, and can be easily discarded.

Surveys

Method of collecting qualitative and quantitative data from a desired set of users. Can be distributed to a very large set of users, so statistical significance can be achieved.

Literature Review

Valuable way to tap into existing knowledge gained from previous studies. An efficient strategy for gaining insights, and may alleviate the need to conduct a new study.
